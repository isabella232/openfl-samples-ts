OpenFL Samples (ES6)
====================

## Get Started

### Clone the repository

```bash
git clone https://github.com/openfl/openfl-samples-es6
```

### Choose a sample

```bash
cd openfl-samples-es6/helloworld
```

### Install and start

```bash
npm install
npm start
```
